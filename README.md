# Captain Ads

This is an Android Ads rotator! (draft version)

## Features

* Banner ads rotation.
* Interstitial ads rotation.

## Gradle Dependency

 ``` gradle
dependencies {
    implementation 'com.github.ybq:Android-SpinKit:1.4.0'
 }
 ```
